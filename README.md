# Karobar (کاروبار) 💼

A professional Urdu-English business management application designed for small to medium enterprises. Effortlessly manage your sales, purchases, expenses, and inventory with a bilingual interface.

## Features ✨

- **Dual Language Support**: Seamlessly switch between Urdu and English.
- **Inventory Management**: Track stock levels, cost prices, and sale prices.
- **Sales & Invoicing**: Generate professional invoices and track customer payments.
- **Purchase Tracking**: Manage supplier records and incoming stock.
- **Expense Management**: Log business expenses with categorized reporting.
- **Real-time Reports**: Visualize your business health with profit/loss analytics.
- **Local Storage**: All data stays private on your browser.

## Built With 🛠️

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Recharts** (Reporting)
- **jsPDF** (Invoices)

## Getting Started 🚀

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/karobar.git
   ```
2. Navigate to the project directory
   ```bash
   cd karobar
   ```
3. Install dependencies
   ```bash
   npm install
   ```

### Running Locally
To start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Building for Production
To create a production build:
```bash
npm run build
```
The optimized files will be in the `dist/` folder.
